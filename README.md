# backend
1. 将后端仓库克隆到本地

   ```bash
   git clone https://github.com/HarmonyAgriLab/backend.git
   ```



2. 创建虚拟环境并指定python版本

   ```bash
   conda create -n your_conda_env_name python=3.12
   ```



3. 激活创建的虚拟环境

   ```bash
   conda activate your_conda_env_name
   ```



4. 安装所需库

   ```bash
   pip install -r requirements.txt
   ```



5. 启动后端服务

   ```bash
   python server.py
   ```



6. 启动订阅服务

   ```bash
   python subscribe.py
   ```
